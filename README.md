# Land Record Verifier

## Overview
Land Record Verifier is a Java-based application that verifies land ownership details using Survey Number. It uses JDBC to connect to a MySQL database and retrieves land records securely.

## Features
- Verify land records using Survey Number
- Java Swing GUI
- JDBC database connectivity
- MySQL database
- Simple and user-friendly interface

## Technologies Used
- Java
- JDBC
- MySQL
- Eclipse IDE

## How to Run
1. Import the project into Eclipse.
2. Create the MySQL database and required tables.
3. Update the database username and password in `DBConnection.java`.
4. Run the application.
